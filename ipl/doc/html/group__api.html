Detailed Description

The following API functions are used to access the IPL functionality. To use Ipl_Prog(), the function Ipl_EnterProgMode() needs to be called first to set INIC in programmming mode. After programming, calling the function Ipl_LeaveProgMode() will reset the INIC and restart it in normal mode.

Performs job by using the referred IPF data. INIC needs to be set in programming mode first (by calling Ipl_EnterProgMode()).

Parameters
jobJob to be performed. All possible jobs are listed here: Jobs
lDataLength of complete IPF data (file size in bytes).
For the job IPL_JOB_READ_FIRMWARE_VER, the parameter can be random (no IPF data required).
pDataPointer to byte array where the IPF data or the first data chunk of the IPF data is stored.
For the job IPL_JOB_READ_FIRMWARE_VER, the pointer can be NULL (no IPF data required).
